# MindLink · 管理端使用手册（Beta）

> 本文给代运营公司的**管理员 / PM / 策划 / 创作者 / 投手**使用。
> 客户端（小程序）的指引见 `client-guide.md`。

## 0 · 5 分钟入门

1. 浏览器打开 https://app.mindlink.example.com
2. 用注册时的手机号 + 密码登录
3. 看到 *管理驾驶舱* → 当天的客户红绿灯 / 团队产能 / 本月指标 / 现金流 / 今日 3 件决策
4. 左侧菜单按角色显示：
   - 管理员：全部
   - PM：客户 / 合同 / 项目 / 月报 / 续约
   - 策划：线索 / 诊断 / 方案 / 项目
   - 创作者 / 投手：项目 / 视频 / 内容生产 / 数据录入

## 1 · 主线流程速查

### 1.1 新客户从 0 到签约（S1 → S3）

1. **线索池**：录入或导入线索（手机号必填）
2. **客户详情**：点开客户 → *开始诊断* → 填 4 把刀 + 4 张卡
3. **生成报告** → AI 出诊断初稿 → 编辑后 *完成诊断*
4. **方案** → 系统按预算推荐套餐（3 选 1）→ 计算报价 → 编辑定位书
5. **定稿** → 客户签字 → 状态自动 `signed`

### 1.2 签约 → 交付（S4 → S5）

1. *合同列表* → 新建合同（基于已签方案）
2. *发起电子签* → 客户在小程序签字
3. 启动会 → *定稿* 自动批量派任务到 PM / 策划 / 创作者
4. 视频看板 7 列状态：策划 / 拍摄 / 剪辑 / 待审 / 已审 / 待发布 / 已发布
5. 客户在小程序审核（通过 / 小改 / 重拍）

### 1.3 复盘 + 续约（S6 → S7）

1. 月初 *数据录入* → 上月各视频播放 / 互动 / ROI
2. *月度报告* → AI 生成 6 段初稿 → PM 30 分钟编辑 → 推送
3. 客户读完月报自动弹 NPS
4. 合同到期前 30 天 → *续约预警* 自动出现
5. *生成续约提案* → 谈判记录 → 标续约成功 / 失败

## 2 · 关键功能

### 2.1 管理驾驶舱

每日 9:00 开屏，5 个模块全景：

- **客户红绿灯**：85+ 绿、60-84 黄、<60 红 · 点红灯客户跳详情
- **团队产能**：4 个角色任务占用率 · ≥95% 红色预警
- **本月指标**：新签 / 续约 / 流失 / 续约率 / 客单价
- **现金流**：本月已收 / 成本 / 净利润
- **今日 3 件决策**：系统自动识别 · 一键跳处理

### 2.2 AI 内容生产

- AI 文案：钩子 + 主体 + CTA 三段输出
- AI 标题：5 候选 + 预期点击率
- AI 标签：平台 + 行业 + 本地 + 热点 15 个
- 老板口播：小程序提词器 + 分段录制
- 配额：基础版 5 万 token / 专业版 20 万 / 企业版 100 万 每月

### 2.3 案例库

- 官方库（思链运营）+ 本租户私库
- 文案 / 标题 / 标签 / 投放 4 子库
- 调用次数 → 热度排序

## 4 · 常见问题

**Q：忘记密码？**
A：登录页 *忘记密码* → 输入手机号 → 短信验证码重置。

**Q：客户老板没收到月报推送？**
A：先看客户在小程序是否绑定过手机号；未绑定则通过 *客户详情 → 通知* 一键发送绑定链接。

**Q：合同状态卡住不动？**
A：*合同详情 → 状态机* 看到当前状态 + 允许跳转。如电子签回调没回来，联系 on-call 手动推。

**Q：AI 输出不满意怎么办？**
A：调高 temperature 重生成；或在 *案例库* 找历史爆款做参考；最终所有内容都可手动编辑。
