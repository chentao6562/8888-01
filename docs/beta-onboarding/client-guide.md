# MindLink · 客户端使用指南（一页图文）

> 给被代运营服务的**老板**们用。10 秒上手。

## 怎么打开

1. 微信扫一扫 → 搜索 *MindLink* 小程序
2. 点 *微信一键登录* → 首次绑定时输入和代运营公司预留的同一手机号
3. 进入 → 看到本月数字 + 待办

## 4 件事

| Tab | 你能做什么 |
|---|---|
| **首页** | 一眼看到本月播放、流水、ROI；下方是待你处理的事 |
| **待我审** | 代运营拍好的视频在这等你 → 点开看 → *通过 / 小改 / 重拍* |
| **月报** | 每月 1 号收到推送 → 读完打个分（满意度 0-10）|
| **续约** | 合同到期前 30 天自动出现 → *预约沟通* PM 会主动找你 |

## 视频审核 60 秒

1. 点 *待我审* → 选一条
2. 播放器播视频
3. 想评论？暂停 → 输入 *在 00:12 插入批注* → 写感想
4. 看完 → 拉到底 → 选 **通过 / 小改 / 重拍**
5. 完成

> 小改 = 改文案改剪辑（同一条视频改完再发）
> 重拍 = 整段重来（拍摄团队会重新约时间）

## 月报 30 秒

1. *月报* tab → 选最新一份
2. 6 段（总览 / 交付 / 流量 / 爆款 / 反思 / 下月重点）滑下去看
3. 系统自动弹出 *满意吗？0-10*
4. 评分 + （可选）写句话 → 提交

## 付款 + 发票

1. *首页 → 待办 → 待付款* 点开
2. 看到分笔款（20% 策划 / 40% 拍摄 / 35% 剪辑 / 5% 尾款）
3. 转账后 → *上传凭证* 拍张照传上来
4. 想要发票 → *申请发票* 填抬头 + 税号 → 1-3 个工作日开出

## 续约

合同到期前 30 天自动收到提醒：
1. *续约* tab 点开
2. 看到 PM 出的方案（绿灯客户有 5% 折扣）
3. 同意 → *预约沟通* → PM 当天联系你
4. 不续 → 直接关闭，PM 会做流失访谈

## 不懂怎么办

- 微信群里 @你的 PM
- 小程序里：*我的 → 联系客服*
