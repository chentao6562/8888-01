# Beta · FAQ

## 账号 / 登录

**Q：注册公司后没收到任何确认？**
A：注册成功即可登录，无需邮件验证（MVP 阶段）。如登录失败，先用 *忘记密码* 重置一次。

**Q：员工怎么加入？**
A：管理员 → *员工管理 → 邀请* → 输入手机号 + 角色 → 系统生成短信邀请链接（Beta 期由管理员复制发给对方）。

**Q：登录失败 5 次被锁怎么办？**
A：30 分钟后自动解锁；急用联系管理员后台手动解锁。

## 客户管理

**Q：客户阶段不能跳到任意状态？**
A：是状态机限制，比如 `lead` 不能直接到 `signed`，必须经过 `diagnosing → proposing`。点 *客户详情 → 状态机* 看允许跳转。

**Q：诊断报告只能 AI 生成吗？**
A：不是。AI 出初稿后可任意编辑，最终内容以人工修改为准。

**Q：方案签字后还能改吗？**
A：不能。已签字 = 已生效。如有变更需新建一个 v2 方案重新走流程。

## 合同 / 付款

**Q：电子签是哪家？安全吗？**
A：法大大（生产）+ MockProvider（Beta 早期 / 演示）。法大大是国内合规电子签头部，签字法律效力等同于纸质。

**Q：必须按 20/40/35/5 分笔吗？**
A：默认是。系统支持自定义比例（合同设置里改），但建议保留默认（"先拍后付" 的核心逻辑）。

**Q：客户上传的付款凭证我在哪里看？**
A：*合同详情 → 付款记录* 每笔下方有 *查看凭证*。

## 内容生产

**Q：AI 调用次数到上限了？**
A：基础版 5 万 / 专业版 20 万 / 企业版 100 万 token / 月。看 *设置 → 用量*。月初自动重置。

**Q：AI 输出包含敏感词？**
A：系统已做敏感词预检 + 后检；如还命中，编辑后再保存。

## 月报 / 续约

**Q：月报客户没看怎么办？**
A：48 小时未读会自动追加一次推送；超过 72 小时通知 PM 电话提醒。

**Q：续约预警出现得太晚？**
A：默认到期前 30 天。可在 *设置 → 续约* 里调到 45 天（限管理员）。

## 数据 / 隐私

**Q：我们的客户数据其他公司能看到吗？**
A：不能。多租户严格隔离 + 行级 `tenant_id` 校验，所有 e2e 都验过。

**Q：数据存在哪？跨境吗？**
A：数据全部在中国境内云（腾讯云 / 阿里云）；AI 调用首选国内大模型（通义 / DeepSeek），不出境。

**Q：注销账号后数据怎么办？**
A：30 天后彻底删除（含备份）；30 天内可联系工单恢复。

## 故障

**Q：页面打不开 / 502？**
A：F5 刷新一次；仍不行 → 飞书群 @on-call。

**Q：客户端登录提示 *未匹配到客户档案*？**
A：客户老板手机号必须和管理端 *客户详情 → bossPhone* 一致。让 PM 改一下后再绑定。

**Q：哪里看自己 SLA 是否被达成？**
A：*管理驾驶舱 → 本月指标 → 续约率 / 流失率* 是核心；可用性 99.5% 由我们保证（看 status 页）。
