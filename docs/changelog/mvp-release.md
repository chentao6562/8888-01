## 三端覆盖

- **管理端 Web**（Vue 3 + Vite）· 30+ 视图 · 像素级还原原型 · 5 角色 RBAC
- **客户端小程序**（uni-app · 微信 + H5）· 7 个页面 · 4 Tab · 老板友好的大字深蓝设计
- **后端 API**（NestJS + TypeORM）· PG / Redis · JWT 双链（管理端 + 客户端独立）

## 安全 + 合规

- AES-256-GCM 列级加密（手机号 / 身份证）
- 5 个登录关键端点限流
- 7 项安全响应头 + HSTS
- CORS 白名单
- 全局审计拦截器 + @Audit 装饰器
- 多租户行级隔离 + RBAC 双重校验
- 隐私政策 + 用户协议页面
- devLogin / Swagger 生产关闭

---

## 部署交付

- Dockerfile · API < 300MB · Web < 50MB（Nginx）
- K8s manifests · Deployment + Service + Ingress + HPA + StatefulSet + CronJob
- GitHub Actions · staging 自动 · prod 手动确认
- 5 篇 Runbook（部署 / 回滚 / 故障 / 备份 / 扩容）

## Beta 启动

### 准入标准

- 本地代运营公司
- 月营收 ≥ 30 万
- 客户数 ≥ 5 家
- 愿意每周 1 次反馈会

### 免费期

3 个月。期间无功能限制，仅按企业版配额（100 万 token / 月）使用 AI。

## V2 路线（基于 Beta 反馈再排序）

### 高优先级候选

- 数据自动采集（接抖音 / 视频号开放平台）

### V3+

- 国际化 / 多语言
- 移动端独立 App
